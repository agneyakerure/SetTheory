# Labels

Where each vibe lives, and what you found there. Plain Markdown — edit it by hand
any time; the app reads it back. `Status` is one of never / dipped / walked.
Anything below the fields is yours to write.

Labels you own tracks from appear in the app automatically, derived from your
Rekordbox colours and your Bandcamp purchases — you don't list them here. This
file is only what you've *said* about them, plus any label you added by hand
because you want to go there.
